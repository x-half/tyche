# 拼豆游戏增强功能需求

## 1. 图片导入转换功能
- 允许用户上传图片（JPG、PNG、WebP）
- 将图片转换为拼豆图案（像素化）
- 支持调整网格尺寸（29x29、48x48、64x64等）
- 支持调整颜色数量（限制为拼豆可用颜色）
- 实时预览转换效果
- 生成颜色用量清单

## 2. 品牌色卡支持
- 支持主流拼豆品牌：Mard、优肯、黄豆豆
- 每个品牌包含完整的颜色库（色号+颜色值）
- 颜色选择器显示品牌色号
- 支持按品牌筛选颜色
- 显示颜色名称和色号

## 3. 进度追踪功能
- 按颜色高亮待拼区域
- 逐板追踪进度（对于大尺寸图案）
- 显示已完成/未完成的珠子数量
- 进度百分比显示
- 支持标记已完成的区域

## 4. 熨烫模拟效果
- 模拟真实熨烫过程
- 珠子熔化动画效果
- 支持单面熨烫和双面熨烫
- 熨烫温度模拟（影响熔化程度）
- 熨烫完成后的效果图

## 5. 保存/加载功能
- 保存当前设计到本地存储
- 加载已保存的设计
- 导出为PNG图片
- 导出为JSON数据文件
- 导入JSON设计文件

## 6. 预设图案库
- 内置常用图案（动物、食物、表情等）
- 按分类浏览图案
- 一键加载图案到画布
- 显示图案难度和所需珠子数量

## 技术要求
- 纯前端实现（HTML/CSS/JS）
- 响应式设计，支持移动端
- 流畅的交互体验
- 真实的视觉效果（珠子质感、阴影、高光）
- 支持撤销/重做操作
