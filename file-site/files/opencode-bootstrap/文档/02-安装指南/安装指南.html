📦 安装指南

OpenCode 的安装非常简单，核心就是一个二进制文件，不依赖任何运行时环境。

方式一：使用本资源包（推荐）

Windows： 双击 [Windows] 双击我安装.bat

macOS / Linux：

chmod +x 脚本/安装.sh
./脚本/安装.sh

方式二：自己安装

# 一行命令安装（推荐）
curl -fsSL https://opencode.ai/install | bash

# 通过 npm
npm install -g opencode-ai

# macOS Homebrew
brew install anomalyco/tap/opencode

验证安装

opencode --version

如果显示版本号（如 1.16.2），说明安装成功。

💡 安装完成后直接输入 opencode 就能用，免费模型已就绪，不需要任何配置。
