# 📦 安装指南

> OpenCode 的安装非常简单，核心就是一个二进制文件，不依赖任何运行时环境。

---

## 安装方式

### 方式一：使用本资源包（推荐）

**Windows：**
```bat
双击 [Windows] 双击我安装.bat
```

**macOS / Linux：**
```bash
chmod +x 脚本/安装.sh
./脚本/安装.sh
```

---

### 方式二：自己安装

**一行命令安装（推荐）：**
```bash
curl -fsSL https://opencode.ai/install | bash
```

**通过 npm 安装（需要先装 Node.js）：**
```bash
npm install -g opencode-ai
```

**macOS 通过 Homebrew：**
```bash
brew install anomalyco/tap/opencode
```

**Windows 通过 winget：**
```bash
winget install opencode
```

---

## 各平台详细步骤

### Windows

1. 打开 PowerShell（管理员）
2. 执行：
   ```powershell
   npm install -g opencode-ai
   ```
3. 输入 `opencode` 启动

如果 npm 不方便，也可以直接去 https://opencode.ai/download 下载安装包。

---

### macOS

```bash
# 一行命令安装
curl -fsSL https://opencode.ai/install | bash

# 或者用 Homebrew
brew install anomalyco/tap/opencode
```

---

### Linux

```bash
# Ubuntu / Debian / CentOS / Arch 都支持
curl -fsSL https://opencode.ai/install | bash
```

安装后可能需要把 OpenCode 加到 PATH：
```bash
echo 'export PATH="$HOME/.opencode/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

---

## 验证安装

```bash
opencode --version
```

如果显示版本号（如 `1.16.2`），说明安装成功。
