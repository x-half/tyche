# 🔧 常见问题

> 遇到问题先看看这里，大部分问题都能解决。

---

## 安装问题

### opencode 命令找不到

**原因：** OpenCode 没有加到系统的 PATH 中。

**解决：**
```bash
export PATH="$HOME/.opencode/bin:$PATH"
# 永久保存
echo 'export PATH="$HOME/.opencode/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc
```

### Windows 上双击没反应

确保你双击的是 `[Windows] 双击我安装.bat`，不是 `.ps1` 文件。
如果还不行，右键点击 → 以管理员身份运行。

### 安装脚本下载失败

检查网络连接，如果在中国大陆可能需要代理：
```bash
export HTTP_PROXY=http://127.0.0.1:7890
export HTTPS_PROXY=http://127.0.0.1:7890
./脚本/安装.sh
```

## 使用问题

### 没有免费模型

在 opencode 中输入：
```
/models
```
如果列表是空的，等几秒再试，或者检查网络。

### AI 回复到一半停了

按 `Ctrl+C` 中断，然后重新输入，或者把问题分小步骤问。

### OpenCode 卡住了

按一次 `Ctrl+C` 取消当前操作。如果不行，再按一次强制退出，然后重新启动。

### 模型太慢

免费模型有一定限制，如果太慢可以：
1. 换个模型试试（输入 `/models`）
2. 等一会儿再用
3. 更简单的任务用轻量模型

## 文件问题

### 配置文件在哪里

| 系统 | 路径 |
|------|------|
| macOS / Linux | `~/.config/opencode/opencode.jsonc` |
| Windows | `%USERPROFILE%\.config\opencode\opencode.jsonc` |

### 想恢复默认配置

删除配置文件，重启 opencode 即可（会自动使用默认设置）：
```bash
rm ~/.config/opencode/opencode.jsonc
```
