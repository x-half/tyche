🔧 常见问题

遇到问题先看看这里，大部分问题都能解决。

opencode 命令找不到

export PATH="$HOME/.opencode/bin:$PATH"
# 永久保存
echo 'export PATH="$HOME/.opencode/bin:$PATH"' >> ~/.bashrc
source ~/.bashrc

没有免费模型

在 opencode 中输入 /models。如果列表是空的，等几秒再试，或者检查网络连接。

AI 回复中断

按 Ctrl+C 中断，重新输入。或者把大问题分成小步骤。

OpenCode 卡住

按一次 Ctrl+C 取消当前操作。不行就再按一次强制退出，然后重启。

模型太慢

  • 换个模型试试（输入 /models）
  • 等一会儿再用
  • 简单任务用轻量模型

系统路径
