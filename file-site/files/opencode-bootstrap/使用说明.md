# 🎯 OpenCode 一站式安装配置资源包

> **装完就能用！** OpenCode 自带免费模型，不需要 API Key，不需要 Node.js，什么都不需要。

---

## 👈 先看这里：你该用哪个文件？

**如果你用的是 Windows：**
> 双击这个文件 → **`[Windows] 双击我安装.bat`**

**如果你用的是 macOS 或 Linux：**
> 在终端中运行这个文件 → **`[Mac-Linux] 运行我安装.sh`**
> ```bash
> bash [Mac-Linux] 运行我安装.sh
> ```

**如果你已经有 Node.js 环境，或者想通过 nvm 安装：**
> Windows → 双击 **`[Windows] 通过nvm安装.bat`**
> Mac/Linux → 运行 **`[Mac-Linux] 通过nvm安装.sh`**

## 🚀 3 步上手

### 第 1 步：解压

把压缩包解压到你喜欢的文件夹。

### 第 2 步：安装

**直接安装（推荐）：**
- Windows → 双击 `[Windows] 双击我安装.bat`
- Mac/Linux → 终端运行 `bash [Mac-Linux] 运行我安装.sh`

**通过 nvm + npm 安装（如果你需要 Node.js 环境）：**
- Windows → 双击 `[Windows] 通过nvm安装.bat`
- Mac/Linux → 终端运行 `bash [Mac-Linux] 通过nvm安装.sh`

### 第 3 步：开始使用

安装完成后，在终端输入：
```bash
opencode
```

> 🎉 第一次打开会自动加载免费模型，直接输入你的问题就开始吧！
